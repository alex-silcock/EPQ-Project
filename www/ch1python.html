What is Python?

Python is a programming language created by Guido Van Rossum in 1991.
It is used for:
  • web development (server side)
  • software development
  • maths
  • artificial inteligence & machine learning
Why would you chose to use Python?
  • It works on many different platforms, for example, Windows, Mac etc.
  • It's similar to the English language
  • It allows developers to write programs with less code than other languages
  • It's an interpreted language, meaning that code is executed line by line so finding errors in code can be easy
